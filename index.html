Linear Programming Example

This page illustrates a simple Linear Programming (LP) problem for a procurement scenario, showing how to model and solve it using the PuLP library in Python.

1. Problem Statement

Our small manufacturing company needs two raw materials (A and B) to produce a final product. We have two suppliers (X and Y) for each material, each with specific costs and capacity limits.

  • We must produce at least 1,000 units of the final product each month.
  • Each unit of the product requires 2 kg of Material A and 1 kg of Material B.
  • Supplier X can provide up to 1,600 kg of Material A and 800 kg of Material B.
  • Supplier Y can provide up to 2,000 kg of Material A and 500 kg of Material B.
  • Costs per kg:
    • Supplier X: $10 (A), $20 (B)
    • Supplier Y: $9 (A), $22 (B)

The objective is to minimize the total cost of purchasing Materials A and B, while satisfying demand and capacity constraints.

2. Mathematical Formulation

We define the following decision variables:

  • xXA: kg of Material A from Supplier X
  • xXB: kg of Material B from Supplier X
  • xYA: kg of Material A from Supplier Y
  • xYB: kg of Material B from Supplier Y

Objective (Minimize Total Cost):

Minimize:
  10 * x_XA + 20 * x_XB
+  9 * x_YA + 22 * x_YB

Constraints:

  1. Material A Demand: xXA + xYA ≥ 2 * 1000 (since each product unit needs 2 kg of A)
  2. Material B Demand: xXB + xYB ≥ 1 * 1000 (each unit needs 1 kg of B)
  3. Supplier X Capacity: xXA ≤ 1600, xXB ≤ 800
  4. Supplier Y Capacity: xYA ≤ 2000, xYB ≤ 500
  5. Non-Negativity: xXA, xXB, xYA, xYB ≥ 0

3. Solving with PuLP (Python)

Below is a sample Python script that uses the PuLP library to solve this LP. You can copy this code into a file (e.g., lp_procurement.py) and run it in a Python environment with PuLP installed (pip install pulp).

# lp_procurement.py

import pulp

# 1) Create the LP problem
model = pulp.LpProblem("ProcurementProblem", pulp.LpMinimize)

# 2) Define Decision Variables
x_XA = pulp.LpVariable('x_XA', lowBound=0, cat='Continuous')  # kg of A from X
x_XB = pulp.LpVariable('x_XB', lowBound=0, cat='Continuous')  # kg of B from X
x_YA = pulp.LpVariable('x_YA', lowBound=0, cat='Continuous')  # kg of A from Y
x_YB = pulp.LpVariable('x_YB', lowBound=0, cat='Continuous')  # kg of B from Y

# 3) Objective Function: Minimize Total Cost
# Cost from X: 10 $/kg for A, 20 $/kg for B
# Cost from Y:  9 $/kg for A, 22 $/kg for B
model += 10*x_XA + 20*x_XB + 9*x_YA + 22*x_YB, "TotalCost"

# 4) Constraints

# 4.1 Material Demand
model += x_XA + x_YA >= 2 * 1000, "A_Demand"
model += x_XB + x_YB >= 1 * 1000, "B_Demand"

# 4.2 Supplier Capacities
model += x_XA <= 1600, "X_A_Capacity"
model += x_XB <= 800,  "X_B_Capacity"
model += x_YA <= 2000, "Y_A_Capacity"
model += x_YB <= 500,  "Y_B_Capacity"

# 4.3 Non-Negativity is implied by lowBound=0

# 5) Solve the Model
model.solve()

# 6) Print the Results
print(f"Status: {pulp.LpStatus[model.status]}")
for var in [x_XA, x_XB, x_YA, x_YB]:
    print(f"{var.name} = {var.varValue:.2f} kg")

print(f"Minimum Total Cost = ${pulp.value(model.objective):.2f}")

After running this script, PuLP will report the optimal solution (amount of Material A and B to buy from each supplier) and the minimum total cost.

4. How to Run the Code

  1. Install Python (3.x) if you haven’t already.
  2. Install PuLP: pip install pulp
  3. Copy the above code into a file named lp_procurement.py.
  4. Open a terminal or command prompt, navigate to the folder containing lp_procurement.py, and run: 
    python lp_procurement.py
  5. Check the terminal output for the solution status, decision variable values, and the minimum total cost.

That’s it! You have solved a simple linear programming model for procurement optimization.
